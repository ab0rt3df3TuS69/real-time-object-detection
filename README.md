# Real-Time Object Detection using YOLOv8

## Overview

This project implements a real-time object detection system using OpenCV and a pretrained YOLOv8 model.

The objective is to study the complete computer vision pipeline, including frame acquisition, image preprocessing, feature extraction, object detection, and performance analysis.

---

## Project Pipeline

Webcam
↓
Frame Acquisition
↓
Image Preprocessing
↓
YOLOv8 Inference
↓
Bounding Boxes
↓
Class Labels
↓
Confidence Scores
↓
Performance Evaluation

## Features

* Real-time webcam capture

* Gaussian blur

* FPS monitoring

The system successfully detected multiple objects in real time and was evaluated under different resolutions and lighting conditions.
