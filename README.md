# Real-Time Object Detection using YOLOv8

## Overview
This project performs real-time object detection using a webcam feed and the YOLOv8 model.

## Features
- Live webcam detection
- Bounding box visualization
- FPS monitoring
- Object class prediction

## Technologies
- Python
- OpenCV
- Ultralytics YOLOv8

## Run
python yolo_webcam.py
